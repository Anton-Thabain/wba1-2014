Was ist Javascript? Eine Einführung

  • Eine Skriptsprache
  • Unterschied zwischen Skript- und Programmiersprache?
  • Gängige Webbrowser können Javascript interpretieren und den Code ausführen (Lynx nicht ;)
  • Javascript lässt sich in HTML Dokumenten einfach integrieren und kann dessen Struktur und aussehen zur Laufzeit verändern
  • Wird sowohl Serverseitig als auch Clientseitig verwendet

Basiert auf dem EcmaScript5 Standard (Release 2011, 6 & 7 sind WIP)

ECMA = European Computer Manufacturers Association (u.a. Microsoft)

Entwickelt von Brendan Eich (Netscape), am Anfang hiess es noch Mocha, LiveScript

Wer mal eintauchen möçhte: https://github.com/lukehoban/es6features

Syntax

  • Literare (Numerische, Strings, Arrays, Objekte)
  • Case Sensitive IF vs. if
  • Klammern und Hochkommata müssen gesetzt werden
  • Bindestriche sind Minuszeichen (Unterstriche stattdessen)
  • Umrbüche, Leerzeichen - ziemlich unempfindlich
  • Das Semikolon ; beendet eine Javascsript-Anweisung
  • Kommentare - Einzeilig // oder Mehrzeili /* */
