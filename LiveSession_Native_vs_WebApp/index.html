Welche ist die Web App, bzw die Native App ?

Native Entwicklung

  • Performance ist gut (+)
  • Monetarisierung durch AppStores (+)
  • InHouse-Entwicklung möglich (+/-)
  • Kosten (iOS) (-)
  • nur ein Zielsystem (-)
  • Reglementierung für Zulassung (-)
  • Gebühren für Developer-Account (-)
  • Berücksichtigen der Programmiersprachen

Web Entwicklung

  • einfacher Einstieg durch HTML5 / CSS3 / JS (+)
  • schnelle Updates (+)
  • keine Regeln für Zulassung (+)
  • „free as free beer“ (+)
  • Performance-Schwierigkeiten je nach App (-)
  • Offline-Daten (-)
  • kein AppStore (-)
  • Hardwarezugriff (-)

Was nutzen wir also am besten ?

  • native Power vs. HTML (lange Listen, flackern, keine volle Bildschirmgröße)
  • monetarisierung der App vs. freies Deployment (schnelle Updates)
  • Zugriff aus Sensoren vs. HTML5
  • nur ein System vs. jedes System native Ui vs. eigene- / Framework-UI SQL-Datenbanken vs. local Storage
